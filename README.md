## QUICK START
### PREREQUISITES
- [Clarinet](https://github.com/hirosystems/clarinet) - Stacks smart contract development tool
- [Node.js](https://nodejs.org/) v16+ - JavaScript runtime
- [Git](https://git-scm.com/) - Version control

```bash
# Clone the repository
git clone https://github.com/your-username/waste-collection-payment-pay-only-after-verified-g.git
cd waste-collection-payment-pay-only-after-verified-g

# Install dependencies
yarn install

# Verify installation
clarinet check

# Run contract checks
clarinet check

# Run tests
yarn test

# Build for production
yarn run build
```
